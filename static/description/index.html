# Kenyan Pharmacy POS ## Overview Complete Point of Sale system specifically designed for pharmacies in Kenya, with full compliance to Kenya Pharmacy and Poisons Board (PPB) regulations. ## Features ### Core Functionality - **Prescription Management**: Complete prescription workflow with validation and tracking - **Drug Registration**: PPB registration number tracking and expiry monitoring - **Patient Records**: Comprehensive patient database with medical history - **Controlled Drugs Register**: Automatic logging of all controlled substance transactions - **Batch & Expiry Tracking**: Full lot/batch tracking with expiry date management - **Pharmacist Verification**: Multi-level approval system for prescription items ### Compliance - Kenya Pharmacy and Poisons Board (PPB) compliant - Controlled drugs register as per regulations - Prescription tracking and archiving - Prescriber verification system - Expiry date monitoring and alerts ### Business Features - Insurance claims processing - Patient co-pay calculations - Multi-payment methods (Cash, M-Pesa, Insurance) - Comprehensive reporting - Stock management with expiry alerts - Real-time inventory tracking ### User Management - Role-based access control - Pharmacy Cashier - Pharmacy Technician - Pharmacist - Pharmacy Manager ## Installation 1. Place the module in your Odoo addons directory 2. Update the app list 3. Install "Kenyan Pharmacy POS" 4. Configure your POS settings under Point of Sale > Configuration ## Configuration ### Initial Setup 1. Create your pharmacy products with PPB registration numbers 2. Add prescribers and verify their credentials 3. Register patients as they visit 4. Configure POS settings for pharmacy-specific features ### POS Configuration - Enable pharmacy POS features - Set prescription validation requirements - Configure expiry date alerts - Set up insurance parameters ## Usage ### Processing Prescriptions 1. Select patient or create new patient record 2. Enter or scan prescription 3. Pharmacist verification 4. Add prescription items to cart 5. Process payment ### OTC Sales 1. Select products 2. Add to cart 3. Process payment (no prescription required) ### Controlled Drugs All controlled drug transactions are automatically logged in the register with: - Patient information - Prescriber details - Pharmacist approval - Transaction timestamp ## Reports - Prescription Reports - Controlled Drugs Register - Expiry Alert Reports - Sales Reports - Stock Reports ## Support For support and customization: - Email: support@softlink.co.ke - Website: https://www.softlink.co.ke ## License LGPL-3 ## Credits Developed by SoftLink Solutions Specialized in healthcare management systems for the Kenyan market
